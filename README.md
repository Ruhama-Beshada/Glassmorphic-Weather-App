# 🌤️ Glassmorphic Weather App

A minimal, modern weather app built with HTML, CSS (Glassmorphism UI), JavaScript, and a tiny Node.js/Express backend. Check the weather for any city in the world with a sleek, frosted-glass dashboard interface.

## 🚀 Features

- ✨ **Glassmorphism UI**: Modern frosted-glass look with smooth animations  
- 🌍 **Global weather**: Get real-time weather for any city  
- 📊 **Key stats**: Temperature, humidity, weather condition  
- ⚡ **Minimal backend**: Node.js + Express API fetching data from OpenWeather  
- 🎨 **Responsive & interactive**: Hover effects, subtle floating cards, fade-ins  

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Glassmorphism + animations), Vanilla JavaScript  
- **Backend**: Node.js, Express  
- **API**: OpenWeatherMap API  
- **Dependencies**: cors, dotenv, node-fetch  

## 🔧 Installation

1. **Clone the repo**
```bash
git clone https://github.com/Ruhama-Beshada/Glassmorphic-Weather-App.git
cd glassmorphic-weather-app
```
2. **Install dependencies**
  ```bash
 npm install
```
3.**Set your OpenWeather API key**
Create a .env file in the root:
API_KEY=YOUR_OPENWEATHER_API_KEY
4.**Run the server**
```bash
node server.js
```
